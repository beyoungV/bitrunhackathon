"# bitrunhackathon" 
打造私钥打碎加密和项目管控平台

现存问题：
1.私钥丢失无法回复
2.私钥被盗
3.投资财产不可控
4.信息缺失

解决方案：
1.私钥打碎分发：按份额打碎、数学算法
2.私钥加密存储：rsa4096、rsa65536
3.钱包管理：转账、收款
4.项目上线：一键发币、技术高级定制
5.项目动态审核及查阅：项目进度、项目花销、投票及审核
6.社群整合：持币入群、好友互动

项目模块简介：
1.个人钱包管理：私钥管理、资产工具、好友管理、转账收款
2.项目投资管理：投资项目、知情查阅、投资权益、资产管理
3.项目管理：融资预备、开启融资、支出申请、钱包管理、社群管理
